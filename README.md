# resume

This repo serves Tran Zha's resume from `tranzha.md`.

Run `node tools/build_web_resume.js --pdf` to regenerate `index.html` and `tranzha.pdf`, then verify both contain the markdown resume text.

Run `node tools/build_upwork_cv.js --pdf` to regenerate the Upwork-focused `tranzha-upwork.html` and `tranzha-upwork.pdf`.
